# Water_Analyzer

Эта программа предназначена для анализа и прогноза потребление воды и ее слива в очисной септик 
Программа позволяет загружать данные из Excel/csv файла (время, активность, объём воды) и выводить  
ключевые статистические показатели и 8 типов графиков для визуализации этих данных

===============================================================================
# Инструкция по установке
===============================================================================
1.Установите Python (версия 3.8 или выше)
ссылка: https://www.python.org/downloads/
обязательно выберите опцию при установке "Add Python to PATH"

Если у вас уже есть установленный Python проверьте версию в командной строке (CMD):
python --version

1.2(опционально) Установите Git (необходим для клонирования репозитория с GitHub)
ссылка: https://git-scm.com/downloads

2.Скачивание приложения
Необходимо создать папку под проект, рекомедуется расположить ее в папке документы
Через командную строку переходим внее с помощью команды:
cd C:\Users\Имя_Пользователя\Documents

Создаем папку проекта и входим в нее через команды:
mkdir Название_папки
cd Название_папки

Клонируем репозиторий через команду:
git clone https://github.com/VladimirWobbly/Sraphic.git
(Или возможно скачивание файлов и их перенос вручную)

3.Создание виртуального окружения
Окружение создается с помощью команды:
python -m venv Название_окружения

Активируем окружение командой:
Название_окружения\Scripts\activate

При успехе в начале командной строки появится (Название_окружения)

4.Установка библиотек 
Для установки библиотек в окружение используется команда:
pip install -r requirements.txt

Дожидаемся полной установки

5.Запуск программы
Для запуска используется команда:
python Water_Analyzer.py

===============================================================================
# Руководство по использованию
===============================================================================
1. Для начала работы загрузите данные в программу, используя кнопку "загрузить Excel файл"
Выберите таблицу с данными (тестовый файл размещен в репозитории git) 

2. Загружаемые данные должны быть формата .xlsx или .csv
Таблицы должны состоять из 3 столбцов время_дата, тип_активности, обьем_воды

3. Для анализа данных необходимо выбрать столбцы для трех оценок соответственно

4. Обратите внимание, для вывода статистики должен быть выбран числовой столбец исходных данных

5. Для вывода результатов используйте соответствующие кнопки в разделах "Анализ данных" и "Графики" 
===============================================================================
# Описание функционала
===============================================================================

1.Статистический анализ
Выводит основные статистики по загруженным данным, вывод квартили и выбросы, меры отклонения

2.Корреляционный анализ
Определение коэффициента корреляции Пирсона, вывод статистической значимости

3.Плотность распределения
График оценки плотности распределения обьемов сливаемой в септик воды, сглаженнаяоценка плотности,
медиана, мода, среднее

4.Гистограмма
Оценка количества итераций слива воды по обьему, средняя к графику

5.Box Plot (IQR)
Выводит разброс основной массы данных между 1 и 3 квартилями

6.Box Plot (Mean ± Std)
Разброс данных от среднего значения, стандартное отклонение, разброс всех точек данных

7.scatter plot
Вывод взаимосвязей обьема сливаемой воды от времени,точечное распределение пар значений

8.Linear Regression
Строится линейная регрессия зависимости обьема воды от времени. Предсказывается значения слива по 7ми предыдущим
Обучение 80%, тестирование 20 % (модель недает успешных предсказаний)

9.Random Forest
Создает сто ветвей решений, обучающихся на рандомной выборке,делает прогноз на следующий день по среднему решению 
(не точная, но более успешная модель)

10.Decision Tree
Реализует прогнозирование по одному древу решений по условням изменений обьяма слива воды
(не точная модель, но дающая неусредненные результаты)
